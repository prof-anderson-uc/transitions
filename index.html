Transition Syntax Reference:

All Transition Properties:

  • transition-property:
  • transition-duration:
  • transition-timing-function:
  • transition-delay:

Shorthand Format:

transition: property  duration  timing-function  delay;

Timing Functions

linear
Consistent speed from beginning to end.
ease
Starts slowly, accelerates, then slows back down at the end.
ease-in
Starts slowly, then speeds up.
ease-out
Starts fast, then slows down.
ease-in-out
Starts slowly, speeds up, then slows down again at the end. Similar to ease, but with less acceleration in the middle.

Transform Options

scale(x)
Increases or decreases the size of the element.
Syntax: .5=half size, 1=current size, 2=twice current size
rotate(x)
Rotates an element clockwise (postive number) or counter-clockwise (negative number) in angled degrees.
translate(x, y)
Moves an element from its current position according to the parameters given for the X-axis and the Y-axis.
skew(x, y)
Skews an element along the X and/or Y-axis by angled degrees.
rotate3d(x,y,z)
Rorates an element along the x (horizontal) y (vertical) and z (depth) axes by degrees.
